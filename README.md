# Audio Player

Audio Player is a Python program for playing mp3 and wav files

## Features

* Play/Pause/Stop songs
* Audio Trimming
* Finding latest songs on billboards
* Next/Back songs
* Shuffle
* Creating/Importing playlists
* Turning mp3 into wav files for pygame
